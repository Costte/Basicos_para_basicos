A continuación, te muestro uno de los códigos de causa más comunes
*Si la llamada dio tono de ocupado se mostrará el puerto por donde salió al lado del texto Telephony y el código (user busy (17)). Esto quiere decir que al número que se marcó esta con una llamada.
*Si la llamada no pudo encontrar una troncal funcional se mostrará el código (no circuit (34)). Esto quiere decir que tenemos fallo en la troncal y requiere una revisión de cableado.
*Si la llamada no es contestada por el número destino se mostrará el código (no answer (19)).
*Si la llamada es rechazada por la tróncales se mostrará el código (network congestion (38)).
*Si el Gateway no sabe cómo enrutar la llamada se mostrará el código (Unassigned number (1)).
